Kawaii Avatars

Avatars now use react-kawaii React components. They render as inline SVGs but require React to display.

To preview avatars, run the dev server:

cd client && npm run dev
# Open http://localhost:5173, create/join a room
# The avatar picker will show all 9 characters

Characters: Boo (Cat), Casper (Ghost), Saturn (Planet), Sundae (IceCream), Mocha (Mug), Scout (Backpack), Chatter (SpeechBubble), Truffle (Chocolate), Tab (Browser)
